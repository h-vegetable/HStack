# 作品集门户设计风格规范

请为我设计一套作品集门户的设计风格页面，用作 Mock 平台、看房日志系统、组件中心等多个应用的统一入口和组件规范。

**整体风格定位：** 轻科幻 · 活力光感

**关键词：** 全息科技 · 玻璃拟态 · 微光动效 · 流线光轨 · 圆润友好 · 通透轻快

> ⚠️ 必须杜绝工业数据平台那种暗沉、厚重、冷静的沉稳风。

> 所有动效和装饰都应呈现 **"有生命的科技感"**——流线轻盈流动、光点柔和呼吸、界面仿佛被注入了轻盈的能量，而非静止的屏幕。

请基于该风格输出设计页，至少包含：

- 浅色 / 深色两套配色的色彩规范
- 全局动态流线装饰与微交互规范
- 按钮、输入框、卡片、对话框、表格、抽屉等常用组件的样式设计（含流线细节）
- 一个作品集门户首页的示例排版

## 一、浅色主题配色方案

> **整体氛围：** 明亮、轻盈，像清晨覆有薄光的科技实验室，带一点俏皮的霓虹感与若隐若现的光流，但不刺眼。

| 属性          | 规范                                                                                                     |
| ------------- | -------------------------------------------------------------------------------------------------------- |
| 主背景        | 微紫调的暖白 → 极淡的冰蓝渐变（`#F5F3FF` → `#EDF2FF`），表面有极低透明度的流动光丝，如同微风拂过的数据流 |
| 卡片/面板背景 | 半透明白色 + 轻微磨砂（Glassmorphism），边缘有极淡的蓝紫色光晕，并有一条极细的渐变光边在缓慢游走         |
| 主色          | 电光紫蓝 `#5D5FEF`，用于主要按钮、选中态、链接                                                           |
| 辅色          | 明亮青绿 `#00D2FF`、活力珊瑚粉 `#FF7A9E`，用于强调、图标高光、标签及流线光点                             |
| 文字          | 深灰蓝 `#1E1E2F`，正文用中灰 `#4A4A6A`                                                                   |
| 阴影与发光    | 柔光阴影，蓝紫光外溢，**不出现硬黑影**；流线经过处会短暂增亮                                             |
| 背景点缀      | 极低透明度的点阵网格、光流线条，以及随机飘浮的微光粒子，粒子带拖尾                                       |

## 二、深色主题配色方案

> **整体氛围：** 深邃的星夜底色，辅以轻霓虹色彩，动态光流如同穿梭在全息投影中，年轻、活泼，并非压抑的暗黑。

| 属性          | 规范                                                                                    |
| ------------- | --------------------------------------------------------------------------------------- |
| 主背景        | 深蓝紫渐变（`#0F0A1E` → `#1A1035`），带有微妙星空噪点和缓慢移动的光带，如遥远星系的光迹 |
| 卡片/面板背景 | 半透明深紫灰 + 强模糊效果，边缘有青紫渐变光边持续流转                                   |
| 主色          | 电光蓝紫 `#8582FF`，用于主操作                                                          |
| 辅色          | 亮青 `#3AE0D9`、暖粉橙 `#FF6B8B`，用于悬停高亮、标签、角标及流线高光                    |
| 文字          | 近白淡紫 `#EDE9FF`，次要文字淡灰紫                                                      |
| 阴影与发光    | 弥散光晕，按钮悬停泛出暖色光，界面边角有扫描光，流线拖尾可见微粒子                      |
| 背景点缀      | 细密星点、动态光粒、偶尔扫过的全息光栅，流线会以波浪或螺旋轨迹轻柔运动                  |

## 三、全局动态效果与流线装饰

> 请将以下动态语言贯穿整个界面，作为风格基因的一部分，而非附加效果。所有动态都应 **平滑、轻量、60fps 友好**，营造"呼吸感"与"能量流动"。

### 光流线条

界面上存在半透明的光带，以非常缓慢的速度沿曲线或直线流动。它们可以穿过卡片、环绕按钮，或在空白区域游走。颜色取自主色到辅色的渐变，透明度约 `5%-15%`，带有一点发光模糊。深色主题下光流更加明显，浅色主题下更为含蓄。

### 粒子系统

背景中随机分布着细小的光点（`2-4px`），它们缓慢飘浮，偶尔汇聚成短流线，或随着鼠标位置轻微偏移。光点色彩可在主色、青绿、粉橙之间柔和变换。

### 扫描与描边光

卡片或面板的边缘，有一条极细的光线（`1-2px`）沿着边框不间断地滑行，如同一次温柔的全息扫描。速度缓慢，周期约 `4-6` 秒。鼠标悬停时扫描速度加快并增亮。

### 呼吸光效

关键交互元素（如主按钮、当前选中项）自带柔和的明暗呼吸，光晕以 `2-3` 秒周期微微扩张收缩，模拟生命体般的律动。

### 转场与出现

新元素出现（如对话框、抽屉）时，伴随一道光流从中心向外或沿滑动方向扫过，随即淡入常态。消失时反向收缩，光流吸回元素内。

### 光标与输入

文本输入光标是一条发光的细竖线，带有同色光晕微呼吸。点击输入框时，边框会有一道流光从点击处快速绕框一周。

## 四、常用元素设计风格（含流线细节）

请将以下组件统一纳入深浅两套主题，并保持一致的动效与流线互动。

### 按钮

- **形状：** 胶囊型或大圆角矩形，圆角 `8–20px`，手感顺滑
- **主按钮：** 使用主色渐变，表层轻微玻璃质感，悬停时向外漫射同色系光晕并略微上浮，同时表面有一条纤细的光泽斜向划过
- **次级按钮：** 幽灵按钮（半透明描边），边框带微弱呼吸发光，悬停时填充薄光，边框流转的光线速度加快
- **危险/删除按钮：** 使用暖色珊瑚粉，暗色主题下泛出柔和霓虹光，点击时扩散出一圈微光涟漪
- **通用：** 所有按钮点击有细微弹性缩放，**禁止厚重投影**

### 输入框

- **外观：** 圆角 `12px`，边框 `1.5px` 半透明主色，背景微透明
- **默认态：** 边框带有极淡呼吸光效，暗色主题下可见微弱内发光，偶尔有流线从一端滑向另一端
- **聚焦态：** 边框变为流动渐变色（主色→辅色），出现一个游走的光点或一道纤细光流，光标也为发光色并脉动
- **占位符文字：** 浅灰紫，字号略小，字体轻细

### 卡片

- **基底：** 磨砂玻璃 + 圆角 `16–24px`，边缘有 `1px` 半透光边，并有一条细光线沿边框持续巡游（扫描光）
- **浅色主题：** 纯白底叠加极淡的蓝紫渐变半透明层，阴影为低透明度弥散蓝紫光，卡片上的光流为淡蓝紫色
- **深色主题：** 深紫黑底，泛出青蓝微光，卡片悬停时亮度提升，光边更明显，扫描光加速，并轻轻浮起
- **装饰：** 可带有细微的科技纹理，如隐约的对角线光纹或角落微标，角落偶尔有粒子飘入

### 对话框 / 模态框

- **遮罩层：** 强模糊 + 暗色半透明（浅色主题用淡蓝灰模糊），遮罩中可隐约看到背景流线缓缓移动
- **对话框本体：** 磨砂玻璃卡片，顶部有一条发光的彩色分割线（可用主色到辅色渐变），也可在角落带一个发光角标。出现时，有一条光流从分割线向两侧扩散开来
- **关闭按钮：** 圆形幽灵按钮，内含一个发光"×"，悬停时整个圆圈泛起光晕并旋转 90°
- **标题：** 字体清晰略宽，可使用渐变文字效果（慎用但可接受）
- **操作按钮：** 居右，符合年轻化应用习惯

### 表格

- **整体：** 无界或轻分割线，行与行之间用极浅的背景色区分
- **表头：** 半透明磨砂条，文字使用主色微光，底部有一条流动的淡光带作为分隔
- **数据行：** 默认透明，悬停时浮现浅色光斑（浅色主题淡紫，深色主题淡青），并在行下方出现一条极短的流线从左侧滑入
- **选中行：** 左侧出现一条 `3px` 发光色带，色带表面光点上下流动
- **滚动：** 表头固定，并微微加深背景
- **斑马纹：** 使用非对称微光条纹，非传统灰色，要透出轻科幻感

### 抽屉（侧边滑出面板）

- **面板背景：** 比卡片更强的磨砂玻璃，边缘带一条流动光带（从顶部滑入时，光带从顶部向底部游动，如同扫描激活）
- **标题栏：** 简洁，左侧可带一个发光的装饰点，标题文字加粗
- **关闭操作：** 一个小巧的发光箭头或返回图标，悬停时箭头周围出现旋转的小粒子
- **内容区：** 与卡片风格一致，列表项悬停浮现微光，并且每个列表项左侧有一条极细的光线在悬停时从上向下延伸
- **动效：** 整个抽屉滑入时带有弹性动效，光带短暂扫过，轻快活泼；关闭时面板轻微缩小并淡出，光流倒吸

## 五、作品集门户首页

- **顶部导航：** 统一磨砂效果固定，带有轻量的全息高光线，导航下方有一条持续流动的细光带作为分隔
- **中心区域：** 展示应用卡片（Mock 平台、看房日志、组件中心），卡片设计使用上述组件风格，配以活泼的科技感图标，图标带有微动效（如缓慢自转、粒子环绕）
- **背景：** 缓慢游动着光粒与蜿蜒的流线，卡片之间偶尔有纤细的光线桥接，暗示数据连通
- **整体调性：** 传达 **"这里是年轻的创造者空间"**，而非严肃的数据监控台
- **载入动画：** 首页载入时，光粒和流线从四周向中心汇聚，最终点亮中央的欢迎语或 logo，随后进入常态漫游
